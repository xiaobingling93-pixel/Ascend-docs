# 蓝区手册普遍问题汇总

整改蓝区手册时关注一下这些问题

1.  大纲问题：蓝区资料大纲不统一，各个手册都略有差别---根据模板统一
2.  图表标题：蓝区文档的截图没有图标题的格式，只能在图片或者表格上方手敲标题，所以一般不给图片或加链接，上下文描述一般用“下图”、“上图”、“下表”、“上表”，如果必须要跳转，那么在图片名称或表标题后面加跳转锚点<n name="figure*"></a>或<n name="table*"></a>
3.  中英文空格问题：蓝区手册普遍中英文之间添加空格---华为资料规范不加空格
4.  界面截图不够细致，未按华为风格规范截图，该模板不做规范---须依照华为风格手册整改
7.  低错：拼写低错、大小写不统一问题、错别字问题、重复字/词问题、描述不清晰问题（语句不通顺、有歧义等）、描述过长等问题---须依照华为风格手册整改，大模型检查已排查过一次，后续整改后需要再用大模型检查一次
