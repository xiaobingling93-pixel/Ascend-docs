# 安装指南

## 安装说明

MindStudio全流程工具链接的安装方式主要有两种：**使用CANN包安装**和**使用子工具run包安装**。

## 使用CANN包安装

以下MindStudio子工具集成在CANN包中，工具安装直接参考[CANN 快速安装](https://www.hiascend.com/cann/download)完成CANN的安装即可；若需要获取子工具代码仓中的最新功能，需要在已安装CANN包的环境下覆盖安装最新的软件包。

| 工具名称 | 子工具安装获取方式                                    |
|:--:| :----------------------------------------------------------|
| msProf | [安装指南](https://gitcode.com/Ascend/msprof/blob/26.0.0/docs/zh/msprof_install_guide.md)                   |
| msPTI | [安装指南](https://gitcode.com/Ascend/mspti/blob/26.0.0/docs/zh/mspti_install_guide.md)                  |
| msMemScope | [安装指南](https://gitcode.com/Ascend/msmemscope/blob/26.0.0/docs/zh/install_guide.md)   |
